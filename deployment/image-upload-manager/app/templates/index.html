🏠 画像アップロード管理システム

顧客が画像をアップロードし、管理者が承認・管理できるシステムです。

📊 管理者ページ 👤 顧客作成 ⚙️ 設定

📋 システム概要

  • 顧客機能: 専用URLから画像をアップロード
  • 管理者機能: アップロード画像の承認・却下
  • 画像種類: 身分証明書、住民票、収入証明書、契約書、銀行明細、その他書類
  • ステータス管理: 未アップロード(×) → アップロード済み(📤) → 承認(○) / 却下(×)

🔗 顧客URLの仕組み

顧客には専用の秘密URLを送信します：

http://localhost:5000/customer/[推測困難な32文字のランダムID]

例: http://localhost:5000/customer/Kx3mN9pQ2rT5vY8wZ1aB4cD6

※ URLを知っている人のみアクセス可能
※ 顧客IDは推測困難なランダム文字列
※ 管理者ページで顧客URLを確認・コピーできます

📚 使用方法

  1. 顧客作成: 「顧客作成」ボタンから新しい顧客を登録
  2. URL共有: 生成された顧客URLを顧客に送信
  3. 画像アップロード: 顧客が専用URLから画像をアップロード
  4. 承認・却下: 管理者ページで画像を確認し、承認または却下
  5. GoogleDrive連携: 承認済み画像は自動的にGoogleDriveにアップロード

⚙️ システム情報

対応画像形式 JPG, JPEG, PNG, GIF
最大ファイルサイズ 16MB
画像処理 自動リサイズ・圧縮
データベース SQLite
